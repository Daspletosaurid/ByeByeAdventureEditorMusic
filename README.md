# Bye Bye Adventure Editor Music
Tired of the *God-forsaken* Adventure Editor theme running on repeat while you spend hours perfecting your best adventure? 
Annoyed when the editor music randomly overlaps your epic music in Test Mode? 

This is the mod for you! It globally mutes the editor music across the entire Adventure Editor, without preventing playback of other music used throughout Spore.

> ℹ️ To install the mod, download the latest `.sporemod` file from the [**Releases**](https://github.com/Daspletosaurid/ByeByeAdventureEditorMusic/releases), and install it using the [Spore ModAPI Launcher](https://launcherkit.sporecommunity.com/) or [SporeModLoader](https://github.com/Rosalie241/SporeModLoader)
